# Credit_Card_Financial_Dashboard-
An end-to-end Data Analytics project that analyzes **credit card customer behavior, financial performance, and transaction trends** using **MySQL** and **Power BI**.

The project transforms raw customer and transaction data into interactive dashboards, enabling business users to monitor key performance indicators (KPIs), identify customer segments, analyze spending patterns, and make data-driven decisions.

📌 Project Overview

This project analyzes 10,108 credit card customers across 28 states for the year 2023.

The analysis focuses on:

- Customer demographics
- Card tier performance
- Revenue generation
- Transaction analysis
- Spending behavior
- Customer satisfaction
- Delinquency analysis
- Business recommendations

🛠️ Tools & Technologies

- MySQL – Database creation and SQL queries
- Power BI Desktop – Dashboard development
- Power Query – Data transformation
- DAX – KPI calculations and measures
- Microsoft Excel / CSV – Dataset

---

📂 Dataset

The project uses two datasets:

1. Credit Card Transaction Dataset

Contains:
- Client Number
- Card Category
- Annual Fees
- Credit Limit
- Transaction Amount
- Transaction Count
- Interest Earned
- Expenditure Type
- Payment Mode
- Quarter
- Week Number

 2. Customer Dataset

Contains:

- Customer Age
- Gender
- Education Level
- Marital Status
- Occupation
- Income
- State
- Customer Satisfaction Score

The datasets are connected using the Client_Num field.

🔍 Key Insights

- Blue card customers account for the majority of revenue but have the highest credit utilization.
- Platinum cardholders generate the highest revenue per customer.
- Businessmen are the highest revenue-generating customer segment.
- Customers aged **40–60 years** contribute the largest share of revenue.
- Graduate-level customers generate the highest overall revenue.
- Bills and Entertainment are the largest spending categories.
- Swipe transactions account for most revenue, while online transactions represent a significant growth opportunity.
- Revenue remains stable throughout all four quarters, indicating a consistent business portfolio.

---

💡 Business Recommendations

- Increase premium card upgrades from Blue to Silver/Gold.
- Improve monitoring of high-utilization Blue card accounts.
- Promote digital payments through online and wallet incentives.
- Focus retention strategies on high-value customers aged 40–60.
- Expand marketing beyond the top-performing states.
- Review underwriting policies for higher-risk customer segments.
